# Project Description

This project demonstrates a basic web server built with Node.js and Express.

The project includes:
- HTTP server
- GET requests
- POST requests
- HTML forms
- Static files
- Public folder
- Image serving

---

# Technologies Used

- Node.js
- Express
- HTML

---

# Features

## GET Routes
The server handles GET requests and returns HTML pages.

Examples:
- `/`
- `/about`

---

## POST Routes
The server also handles POST requests using HTML forms.

---

## HTML Forms
The project contains:
- GET form
- POST form

The forms send data to the server.

---

## Static Files

A `public` folder is used for static files such as:
- HTML files
- Images

Example:
- `cat.png`

---

# Installation

Install dependencies:

```bash
npm install
```

---

# Run The Server

Start the server:

```bash
node app.js
```

---

# Open In Browser

```text
http://localhost:3000
```

Example page:
- `http://localhost:3000/index.html`
